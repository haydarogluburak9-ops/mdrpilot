# Product photos for Declaration of Conformity

## Uploaded product photo

In **Product → Edit**, upload a default product image. It appears in every row of the DoC model table unless a per-model file exists below.

## Per-model photos (optional)

Place PNG/JPEG files in this folder named by **model code slug**:

| Model code in app | File name |
|-------------------|-----------|
| `YM – 15A` | `ym-15a.png` |
| `AT – 30A-R` | `at-30a-r.jpg` |

Per-model files override the default uploaded photo for that row only.

Supported extensions: `.png`, `.jpg`, `.jpeg`, `.webp`
