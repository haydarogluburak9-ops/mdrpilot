# ISO 15223-1 Symbol Pictograms

Place your (licensed) ISO 15223-1 symbol image files in this folder. They are
embedded automatically next to each line of the "Symbols used / Kullanılan
Semboller" part of the Technical File DOCX export.

## Naming

Name each file by its **ISO 15223-1 clause number**, plus two special keys:

| File name      | Symbol                                      | Clause / ref     |
| -------------- | ------------------------------------------- | ---------------- |
| `5.1.1.png`    | Manufacturer / Üretici                      | ISO 15223-1 5.1.1 |
| `5.1.2.png`    | Authorised representative / Yetkili temsilci | 5.1.2            |
| `5.1.3.png`    | Date of manufacture / Üretim tarihi          | 5.1.3            |
| `5.1.4.png`    | Use-by date / Son kullanma tarihi            | 5.1.4            |
| `5.1.5.png`    | Batch code (LOT) / Parti kodu                | 5.1.5            |
| `5.1.6.png`    | Catalogue number (REF) / Katalog numarası    | 5.1.6            |
| `5.2.3.png`    | Sterilized using ethylene oxide (EO)         | 5.2.3            |
| `5.2.4.png`    | Sterilized using irradiation (gamma)         | 5.2.4            |
| `5.2.6.png`    | Sterilized using steam/dry heat              | 5.2.6            |
| `5.2.8.png`    | Do not use if package is damaged             | 5.2.8            |
| `5.2.11.png`   | Sterile barrier system                       | 5.2.11           |
| `5.4.2.png`    | Do not reuse / Tekrar kullanmayınız          | 5.4.2            |
| `5.4.3.png`    | Consult instructions for use                 | 5.4.3            |
| `5.4.4.png`    | Caution / Dikkat                             | 5.4.4            |
| `5.7.7.png`    | Medical device / Tıbbi cihaz                 | 5.7.7            |
| `udi.png`      | UDI carrier                                  | ISO 20417        |
| `ce.png`       | CE marking                                   | MDR Annex V      |

## Notes

- Supported formats: **png, jpg, jpeg, gif** (png recommended).
- Missing files simply fall back to text — no error.
- Square pictograms work best; aspect ratio is preserved automatically.
- These standardized symbols are owned by ISO; use only files you are licensed
  to reproduce.
